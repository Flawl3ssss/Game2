# PlayCanvas Mobile Branch

Эта ветка переносит «Снежный Разгон» на PlayCanvas Engine без необходимости запускать локальный PlayCanvas Editor.

## Страницы

- `index.html` — существующая версия на Three.js.
- `playcanvas.html` — новая мобильная версия на PlayCanvas.

## Что уже реализовано

- управление касанием, мышью и клавиатурой;
- фиксированный пул бесконечных чанков;
- детерминированная перестройка чанков;
- монеты, камни, ускорители и трамплины;
- прыжок и объезд препятствий;
- мобильный HUD и безопасные отступы;
- отдельная сборка Vite для обеих версий;
- автоматическая публикация ветки через GitHub Pages workflow.

## Проверки

```bash
npm install
npm run check
npm run build
```

После сборки PlayCanvas-версия находится в `dist/playcanvas.html`.
