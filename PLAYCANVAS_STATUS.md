# Статус переноса

Текущий этап: первый полностью кодовый вертикальный срез PlayCanvas.

## Готово

- отдельная ветка, не затрагивающая рабочую Three.js-игру;
- npm-зависимость PlayCanvas Engine;
- отдельная Vite-страница `playcanvas.html`;
- объектный пул бесконечных чанков;
- мобильное управление;
- монеты, препятствия, ускорители и трамплины;
- HUD, стартовый экран и экран результата;
- GitHub Actions для проверки сборки и публикации.

## Следующий этап после зелёной сборки

1. Автоматический мобильный smoke-test.
2. Более качественная модель персонажа и санок.
3. Эффекты снега, следы и улучшенное освещение.
4. Экономика и сохранения.
5. Yandex Games SDK.
